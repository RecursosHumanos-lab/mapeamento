## 🚀 Como publicar no GitHub Pages

### 1. Criar o repositório
1. Acesse [github.com](https://github.com) e faça login
2. Clique em **"New repository"**
3. Dê um nome (ex: `mapeamento-cargos`)
4. Deixe como **Public**
5. Clique em **"Create repository"**

### 2. Fazer upload do arquivo
1. Na página do repositório recém-criado, clique em **"uploading an existing file"**
2. Arraste o arquivo `index.html` para a área indicada
3. Clique em **"Commit changes"**

### 3. Ativar o GitHub Pages
1. Vá em **Settings** (no menu do repositório)
2. Role até a seção **"Pages"** no menu lateral esquerdo
3. Em **"Source"**, selecione **Branch: main** e pasta **/ (root)**
4. Clique em **Save**
5. Aguarde ~1 minuto

### 4. Acessar o site
O site estará disponível em:
```
https://SEU_USUARIO.github.io/mapeamento-cargos/
```
